03

为什么需要图神经网络？

传统神经网络假设数据是规则的（序列/网格），但图的拓扑结构是不规则的

传统方法的局限

❌
MLP

忽略图结构，仅使用节点特征，丢失拓扑信息

❌
CNN

要求规则网格结构，无法处理任意拓扑的图

❌
RNN

适用于序列数据，难以处理图中的多对多关系

GNN 的优势

✅
结构感知

直接在图结构上操作，保留拓扑信息

✅
消息传递

节点通过邻居聚合信息，学习图中的模式

✅
置换等变

对节点排列顺序不敏感，符合图的本质特性

GNN 模型架构

一个典型的 GNN 由多层消息传递层堆叠而成，逐层提取更高阶的图特征

Input
Graph G = (A, X)
X ∈ ℝN×din
→
GNN Layer 1
AGG → Linear → σ
H(1) ∈ ℝN×d1
→
GNN Layer 2
AGG → Linear → σ
H(2) ∈ ℝN×d2
→
Task Layer
Classification / Regression
ŷ ∈ ℝN×C
核心公式 — 层间传播
H(l) = σ( D̃-½ Ã D̃-½ H(l-1) W(l) )
Ã = A + IN （加入自环的邻接矩阵） D̃ （Ã 的度矩阵） W(l) （第 l 层的可学习权重） σ （激活函数，如 ReLU）
05

消息传递机制 (Message Passing)

GNN 的核心思想：每个节点通过聚合邻居信息来更新自身表示

Step 0 — 初始化
hv(0) = xv
每个节点的初始特征就是其属性向量
Step 1 — 构造消息
mu→v = W(l) · hu(l-1)
每条边上传递的消息 = 对源节点特征做线性变换
Step 2 — 聚合消息
m̄v = Σu∈N(v) mu→v
将所有邻居发来的消息求和聚合
Step 3 — 更新表示
hv(l) = σ( m̄v + Wself(l) · hv(l-1) )
聚合邻居消息 + 自身特征 → 激活 → 新表示

图卷积网络 (GCN)

Kipf & Welling, 2017 — 将卷积操作推广到图结构数据的经典方法

H(l) = ReLU( D̃-½ Ã D̃-½ H(l-1) W(l) )
D̃-½ÃD̃-½
×
H(l-1)
×
W(l)
→
H(l)
Step 1 — 对称归一化
D̃-½ Ã D̃-½
Ã = A + I（加自环），D̃ 是 Ã 的度矩阵。
归一化防止度数大的节点主导聚合。
Step 2 — 邻居特征聚合
D̃-½ Ã D̃-½ × H(l-1)
归一化邻接矩阵乘以特征矩阵 = 聚合邻居特征。
每个节点得到邻居特征的加权平均。
Step 3 — 线性变换
聚合结果 × W(l)
通过可学习权重矩阵 W 将特征映射到新的维度空间。
这是 GNN 学习的核心参数。
Step 4 — 激活函数
ReLU( · )
对输出施加非线性激活，增强模型表达能力。
ReLU: max(0, x)，输出即为 H(l)。
07

图注意力网络 (GAT)

Veličković et al., 2018 — 通过注意力机制为不同邻居分配不同权重

GCN：固定权重

邻居权重仅由节点度数决定，对所有邻居一视同仁

GAT：自适应注意力

通过注意力机制动态计算每个邻居的重要性权重

GAT 注意力计算流程

1
计算注意力系数
eij = LeakyReLU( aT [W·hi ‖ W·hj] )
2
Softmax 归一化
αij = softmaxj(eij) = exp(eij) / Σk∈N(i) exp(eik)
3
加权聚合
hi' = σ( Σj∈N(i) αij · W·hj )

GNN 应用任务

根据预测目标的不同，GNN 可以解决三类核心图学习任务

节点分类 (Node Classification)

预测图中每个节点的类别标签

论文分类 (Cora) 用户画像 蛋白质功能预测
ŷv = GNN(A, X)v → 分类头 → P(yv | G)

图分类 (Graph Classification)

对整个图进行分类，需要 Readout 操作

分子性质预测 蛋白质结构分类 代码漏洞检测
hG = READOUT({hv : v ∈ V}) → 分类头 → P(y | G)

链接预测 (Link Prediction)

预测图中两个节点之间是否存在边

推荐系统 知识图谱补全 药物相互作用
pij = σ( hiT · hj ) → P(edge(i,j) | G)
